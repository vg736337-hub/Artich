# Artich
Artich Optimizer — легкая утилита на Python &amp; Eel для ускорения Windows. 🚀  Функции:      🛡️ Отключение телеметрии и слежки.      🧹 Очистка Temp-файлов и кэша.      🔑 Активация системы (MAS/KMS).      🎨 Темы: Dark, Cyber, White.      📊 Мониторинг CPU/RAM.  Идеально для слабых ПК. Быстро, просто, эффективно. Твой ПК скажет «спасибо»! 💻✨
